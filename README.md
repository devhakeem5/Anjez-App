### Overview
A powerful and intuitive Todo application built with **Flutter**, designed to help users manage their daily tasks efficiently. The app provides a clean user interface with advanced features for task organization, categorization, and performance tracking.

### Key Features
*   **Task Management**: Create, edit, and delete tasks with ease.
*   **Categories**: Organize tasks into customizable categories with unique colors.
*   **Notifications**: Stay on track with local reminders for upcoming tasks.
*   **Statistics & Analytics**: Visualize your productivity with insightful charts and completion ratios.
*   **Onboarding**: smooth introduction to app features for new users.
*   **Localization**: Full support for both Arabic and English languages.
*   **Theme Support**: Clean and modern design.

### Tech Stack
*   **Framework**: [Flutter](https://flutter.dev) (Dart)
*   **State Management**: [GetX](https://pub.dev/packages/get)
*   **Database**: [SQFlite](https://pub.dev/packages/sqflite) (SQLite)
*   **Local Notifications**: [flutter_local_notifications](https://pub.dev/packages/flutter_local_notifications)
*   **Storage**: Shared Preferences
*   **Utils**: intl, uuid, timezone

### Developer Requirements
To run this project, make sure you have:
1.  **Flutter SDK**: `^3.10.0`
2.  **Dart SDK**
3.  **Android Studio** / **VS Code** with Flutter extensions.
4.  An Emulator or a physical device.

### Getting Started
1.  **Clone the repository**:
    ```bash
    git clone https://github.com/devhakeem5/TODO-List-Flutter.git
    ```
2.  **Install dependencies**:
    ```bash
    flutter pub get
    ```
3.  **Run the application**:
    ```bash
    flutter run
    ```
